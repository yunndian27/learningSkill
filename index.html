CSS 選取器

  1. Tag
  2. class
  3. ID
  4. 屬性
  5. 僞元素(虛擬元素)
  6. 僞類別(虛擬類別)
    • 動態僞類別（:hover、:active...等）
    • 結構僞類別（:first-child、:last-child...等）
    • 迴圈僞類別（:nth-child、:nth-of-type...等）
    • 表單僞類別（:disabled、:enabled...等）

一個選取自由度超高的選取器

  1. 選取【擁有特定屬性】的物件

    寫法是 [屬性]

      <img src="" alt="跳跳虎很可愛">
  <img src="" alt="史努比更可愛">
  <img src="">

      img { border: 10px solid red; }
  img[alt] { border:none; }

    Demo 1

  2. 選取屬性值【完全等於】特定文字的物件

    寫法是 [屬性=值]

      <img src="" alt="跳跳虎是橘色的">
  <img src="" alt="史努比是白色的">
  <img src="" alt="布丁狗是黃色的">

      img[alt="布丁狗是黃色的"] { border:none; }

    Demo 2

  3. 選取屬性值【開頭等於】特定文字的物件

    寫法是 [屬性^=值]

      img[alt^="布丁狗"] { border:none; }

    Demo 3

  4. 選取屬性值【結尾等於】特定文字

    寫法是 [屬性$=值]

      <h2>圖片下載</h2>
  <div class="download-list">
    <a href="images/demo/tiger.jpg"">跳跳虎</a>
    <a href="images/demo/snoopy.jpg"">史努比</a>
    <a href="images/demo/puddingDog.jpg"">布丁狗</a>
  </div>

      a[href$="tiger.jpg"] { background-image: url("images/demo/tiger.jpg") }

    Demo 4 - 圖片下載

    跳跳虎
    史努比
    布丁狗

  5. 選取屬性值中【包含特定文字】的物件

    寫法是 [屬性*=值]

      <img src="" alt="跳跳虎是橘色的">
  <img src="" alt="史努比是白色的">
  <img src="" alt="布丁狗是黃色的">

      img[alt*="布丁狗"][alt*="黃色的"] { display:none; }

    Demo 5

  6. 選取屬性值中【包含特定單字】的物件

    寫法是 [屬性~=值]，作用的方式是『只要屬性值中有指定的單字』，針對中文語系國家來說，這個選取器的效果可能不容易測試出來，但是對於外語來說就會非常明顯了

      <img src="" alt="tiger is cute">
  <img src="" alt="snoopy is Cute">
  <img src="" alt="puddingdogIsCute">

      img[alt*="Cute"] { border-color:yellow; }

      img[alt~="Cute"] { border-color:blue; }

    Demo 6
